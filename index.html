Portal Ibadah Mobile APK

Dokumentasi Resmi & Catatan Rilis

Deskripsi

Portal Ibadah Mobile adalah aplikasi Android native yang diadaptasi dari versi web Portal Ibadah. Aplikasi ini dirancang untuk memudahkan umat Muslim dalam beribadah dengan fitur seperti Al-Qur'an Digital, Jadwal Sholat, dan Kumpulan Doa.

Download & Instalasi

Cara Install file .apk:

  1. Download file APK terbaru dari link yang diberikan.
  2. Buka file tersebut di HP Android Anda.
  3. Jika muncul peringatan keamanan, izinkan instalasi dari "Unknown Sources" (Sumber Tidak Dikenal).
  4. Tunggu hingga proses instalasi selesai.

Catatan Rilis (Changelog)

Versi 1.1.0 (Terbaru)

  • Custom Prayer Alarms: Fitur baru untuk menambahkan jadwal alarm manual. Anda bisa membuat hingga 5 alarm tambahan (misal untuk Sholat Tahajud atau Dhuha).
  • Perbaikan Tanggal Hijriyah: Memperbaiki bug di mana nama bulan Hijriyah tidak sesuai (misal Juli terbaca Rajab). Sekarang menggunakan algoritma kalender yang lebih akurat.
  • Optimasi Izin (Permissions): Perbaikan pada cara aplikasi meminta izin Notifikasi dan Lokasi agar lebih stabil di Android versi baru.

Versi 1.0.0

  • Rilis perdana aplikasi.
  • Fitur: Al-Qur'an 30 Juz, Jadwal Sholat Real-time, Doa Harian.
  • Mode Gelap & Terang.

Informasi Teknis

  • Framework: Next.js 14
  • Engine: Capacitor 5 (Android)
  • Styling: Tailwind CSS
