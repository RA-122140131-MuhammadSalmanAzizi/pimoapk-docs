Portal Ibadah Mobile

Aplikasi mobile all-in-one yang komprehensif untuk kebutuhan ibadah harian Muslim Indonesia. Dibuat dengan teknologi web modern dan dibungkus menjadi aplikasi native Android yang ringan, cepat, dan mengutamakan privasi pengguna.

Dokumentasi ini mencakup fitur, panduan instalasi (APK Master), dan catatan rilis (changelog) yang lengkap untuk setiap pembaruan aplikasi.

Fitur Utama

  • Al-Qur'an Digital: Akses lengkap 30 Juz dan 114 Surah dengan terjemahan Bahasa Indonesia serta audio murottal berkualitas tinggi per ayat dan per surah.
  • Jadwal Sholat Akurat: Perhitungan waktu sholat presisi untuk 500+ kota di Indonesia menggunakan data terpercaya Kemenag/MyQuran API, dilengkapi countdown waktu sholat berikutnya.
  • Kumpulan Hadits: Koleksi hadits pilihan dari Shahih Bukhari disajikan dalam format kartu yang mudah dibaca dan elegan.
  • Doa Harian: Himpunan doa-doa penting untuk berbagai aktivitas sehari-hari.
  • Notifikasi Adzan: Sistem pengingat waktu sholat otomatis berbasis lokasi (Fitur Native Android).
  • Pengalaman Tanpa Iklan: Fokus beribadah tanpa gangguan iklan pop-up atau banner yang mengganggu kekhusyukan.

Technology Stack

Aplikasi ini dikembangkan menggunakan stack teknologi open-source modern untuk menjamin performa dan kemudahan pemeliharaan:

  • Framework: Next.js 16 (React Framework)
  • Styling: Tailwind CSS
  • Native Runtime: Capacitor v6 (Android Bridge)
  • OTA Updates: Capgo Cloud (Live Updates Deployment)
  • Data Sources: eQuran.id, MyQuran API, Hadith API.
